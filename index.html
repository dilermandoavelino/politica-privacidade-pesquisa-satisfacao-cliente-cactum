Política de Privacidade

Esta Política de Privacidade explica como o aplicativo Pesquisa Satisfação Cliente, desenvolvido pela Cactum Systems, trata informações e protege a privacidade dos usuários.

Aplicativo: Pesquisa Satisfação Cliente Desenvolvedor: Cactum Systems Última atualização: 2026

Sumário

1. Coleta de informações 2. Uso da câmera 3. Uso da internet 4. Armazenamento de dados 5. Compartilhamento 6. Segurança 7. Alterações 8. Contato

1Coleta de informações

O aplicativo não coleta dados pessoais sensíveis, não exige login e não solicita cadastro do usuário.

  • Não coletamos nome, CPF, endereço ou dados financeiros.
  • Não vendemos informações dos usuários.
  • Não compartilhamos informações pessoais com terceiros.
As informações registradas no aplicativo são relacionadas somente às respostas da pesquisa de satisfação, como nota, descrição e emoji selecionado.

2Uso da câmera

O aplicativo pode solicitar acesso à câmera exclusivamente para permitir que o responsável pela configuração capture uma imagem para definir a logomarca da empresa.

  • A câmera não é usada para identificar pessoas.
  • A câmera não é usada para monitoramento.
  • A imagem escolhida é usada apenas como logomarca dentro do aplicativo.

3Uso da internet

O aplicativo pode utilizar conexão com a internet para funcionalidades futuras, atualizações, suporte ou possíveis integrações com sistemas e APIs.

Nenhuma informação pessoal sensível é transmitida sem finalidade legítima relacionada ao funcionamento do aplicativo.

4Armazenamento de dados

Os dados da pesquisa são armazenados localmente no dispositivo onde o aplicativo está instalado.

  • As respostas podem ser consultadas no próprio dispositivo.
  • O usuário pode limpar os dados salvos quando desejar.
  • Dados locais podem ser perdidos se o aplicativo for desinstalado ou se os dados do app forem apagados.

5Compartilhamento de dados

A Cactum Systems não vende, aluga ou compartilha dados pessoais dos usuários com terceiros.

As informações registradas são utilizadas apenas para fins de operação e análise da satisfação dos clientes dentro do próprio aplicativo.

6Segurança

Aplicamos boas práticas de desenvolvimento para preservar a segurança das informações armazenadas localmente no dispositivo.

Mesmo assim, nenhum sistema é totalmente imune a falhas, por isso recomendamos manter o dispositivo protegido e atualizado.

7Alterações nesta política

Esta Política de Privacidade poderá ser atualizada para refletir melhorias no aplicativo, mudanças legais ou novas funcionalidades.

A versão mais recente estará sempre disponível nesta página.
